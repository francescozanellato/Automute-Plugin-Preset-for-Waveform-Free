# Automute-Plugin-Preset-for-Waveform-Free
Automute Plugin Preset for Waveform (similar to the discontinued "Ice9" plugin by Cerberus Audio).
This preset works in Tracktion Waveform and in Waveform Free. Tested in Tracktion Waveform Free 13.5.25 (Built: 20 January 2026) (64-bit).

# INSTALLATION
Just drag and drop the file with extension ".trkpreset" on one of your Waveform Tracks (e.g. the Master Track) and the preset will be imported in Waveform as a preset (that can be re-used for any project.

# USE
Just regulate the quick parameter / slider to set the Threshold above which the track will be muted.
Use this preset on the Master Track (or any other track of your choice) to automatically mute that track when the level exceeds the user-defined threshold. It will automatically unmute the track when  the volume gets below the threshold.
Double click the plugin to see al the Plugin Rack details and to set other parameters.
